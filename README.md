# 浙江财经大学图书馆自动化助手

浙江财经大学图书馆图书续期、馆藏查询、到期提醒、座位预约自动化工具。

## 功能介绍
- 自动登录浙财大图书馆
- 自动查询已借图书
- 一键续期所有可借图书
- 到期自动提醒（避免罚款）
- 支持 GitHub Actions 自动运行

## 使用方法
1. 填写自己的学号、统一身份认证密码
2. 开启 GitHub Actions 自动执行
3. 每天自动检查、自动续期

## 学校官网
- 浙江财经大学：https://www.zufe.edu.cn/
- 浙财大图书馆：https://lib.zufe.edu.cn/
